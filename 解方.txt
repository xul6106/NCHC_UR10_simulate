==================== Windows CMD ====================
1. 查詢USB阜號(DualSense)：
-----------------------------------------------------
usbipd list
-----------------------------------------------------

2. 將外部USB綁定、連結到WSL中：
-----------------------------------------------------
usbipd bind --busid 2-2
usbipd attach --wsl --busid 2-2
-----------------------------------------------------

=================== Ubuntu bash =====================

1. 回上一個目錄/回主目錄：
-----------------------------------------------------
cd.. / cd ~
-----------------------------------------------------

2. 刪除多個資料夾(連同底下檔案)：
-----------------------------------------------------
rm -rf <filename>
-----------------------------------------------------
------ 刪除目錄下名為175524xxxxxxxxx的多個檔案 ------
rm -rf 175524*
-----------------------------------------------------

3. 移動檔案

-- 將123.txt移到xul資料夾底下，並重新命名為456.tx ---
mv 123.txt xul/456.txt
-----------------------------------------------------

====================== Python =======================

1.遇到ImportError: Cannot initialize a EGL device display. AttributeError: 'MjRenderContextOffscreen' object has no attribute 'con'
---------- 在程式中加入這行(要import os) ------------
# 強制使用 OSMesa 軟體渲染
os.environ['MUJOCO_GL'] = 'osmesa'
-----------------------------------------------------
